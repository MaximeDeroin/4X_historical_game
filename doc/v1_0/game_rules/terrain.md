# Terrain
Map tiles are composed of a base tile, and can contain additional resources

## Base tiles
* Water 
  *Type: Water
  * _Can only be accessed by boats_
  *Base stats: 
    * 1 Food
* Ground
  * Type: Ground
  * Base stats: 
    * 2 Food
    * 1 Prod
